# Orbiit Tools

This directory contains utility scripts and tools for advanced Wii operations, such as DLC management and WAD conversion.

## Included Tools

### `setup_tools.ps1`
Run this script to automatically download the external tools required for the [Rock Band DLC Guide](../docs/ROCK_BAND_DLC_GUIDE.md).

It will attempt to fetch:
1.  **Wii Mod Lite**: For installing WADs and patching IOS.
2.  **wad2bin**: For converting DLC WADs to SD card format.
3.  **xyzzy-mod**: For dumping keys.

## Manual Downlaods
If the script fails, you can download them manually:
*   [Wii Mod Lite](https://github.com/RiiConnect24/Wii-Mod-Lite/releases)
*   [wad2bin](https://github.com/DarkMatterCore/wad2bin)
*   [xyzzy-mod](https://wiidatabase.de/downloads/wii-tools/xyzzy/)
